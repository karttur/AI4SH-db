This blog contains a draft outline of a PostgreSQL database (schemas and tables) for the AI for Soil Health (AI4SH) work package (WP) on in-situ data. Each post, except the first, contains a different schema, with all the tables and columns defined using Database Markup Language (DBML). The first post is an overview of all the schemas.

All posts (schemas) were updated on 3 September 2024.
